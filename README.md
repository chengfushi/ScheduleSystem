# 日程管理系统

## 项目简介
这是一个基于Java Web技术开发的日程管理系统，帮助用户高效地管理和追踪他们的日常任务和日程安排。

## 功能特点
- 用户注册和登录
- 添加、删除、修改日程
- 标记日程完成状态
- 查看个人日程列表
- 简洁直观的用户界面

## 技术栈
- 后端：Java
- Web框架：Jakarta Servlet
- 数据库：MySQL
- 前端：HTML、CSS、JavaScript

## 安装说明
1. 克隆项目到本地：
```bash
git clone [项目地址]
```

2. 配置数据库：
- 创建MySQL数据库
- 导入项目中的SQL文件
- 修改数据库连接配置

3. 配置开发环境：
- 安装JDK
- 配置Maven
- 配置Web服务器（如Tomcat）

4. 运行项目：
- 使用Maven构建项目
- 部署到Web服务器
- 访问系统首页

## 使用说明
1. 用户注册/登录：
   - 新用户需要先注册账号
   - 使用用户名和密码登录系统

2. 日程管理：
   - 点击"添加"按钮创建新日程
   - 在日程列表中可以修改或删除日程
   - 可以标记日程为已完成状态

3. 查看日程：
   - 登录后自动显示个人日程列表
   - 可以查看所有日程的完成状态
